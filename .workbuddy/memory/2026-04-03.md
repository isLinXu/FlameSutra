## FlameSutra GitHub Pages 在线阅读站点构建完成

### 技术方案
- **SPA架构**: 单页应用，Hash路由 (#vol01, #vol02, ...)
- **Markdown渲染**: 客户端使用 marked.js + DOMPurify 渲染
- **修仙主题**: 暗色背景 + 火焰金渐变 + 境界色彩体系
- **功能**: 全文搜索(⌘K)、侧边栏导航、阅读进度条、TOC目录、上下卷导航

### 文件结构
```
site/
├── index.html          # 主入口（构建时注入页面数据）
├── css/style.css       # 修仙主题暗色样式
├── js/app.js           # SPA路由 + Markdown渲染 + 搜索
└── pages/              # 各卷Markdown内容
    ├── vol01.html ~ vol10.html
    ├── huofire.html, debug.html, dataset.html
```

### 构建命令
- `python build.py` — 构建站点（读取所有README.md，生成pages/和注入index.html数据）
- `cd site && python -m http.server 8080` — 本地预览

### 部署到 GitHub Pages
- Settings → Pages → Source: Deploy from branch
- 选择 site/ 目录（需调整）或使用 GitHub Actions
