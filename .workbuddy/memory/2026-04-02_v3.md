### 第一轮补充内容（已实现）

基于参考资料《AI算法研究员经验与教训手册》和《LLM Engineer's Handbook》，为第二卷添加了**第四章（进阶篇）：天道法则的进化之路**，涵盖四大前沿技术：

1. **Flash Attention** - 闪电般的灵力操控
   - 传统注意力的内存瓶颈分析（seq_len=4096 需要 4GB 显存）
   - 分块计算、融合核、在线 softmax 核心技术
   - 完整的简化版 Flash Attention 实现
   - PyTorch 2.0+ 的 `F.scaled_dot_product_attention` 使用示例
   - 内存优化：O(S²) → O(S×d_k)，节省 20 倍显存

2. **RoPE（旋转位置编码）** - 相对位置感知
   - 绝对位置编码的外推性问题分析
   - 复数旋转的数学基础和群性质证明
   - 高维 RoPE 的完整实现（预计算频率、复数形式应用）
   - 外推能力对比：RoPE 远优于正弦编码
   - 零参数优势：完全解析式，无需训练

3. **GQA（分组查询注意力）** - 计算效率的极致
   - MHA、MQA、GQA 的权衡分析
   - KV Cache 内存占用计算（32 层模型：2GB → 512MB，节省 75%）
   - 完整的 `GroupedQueryAttention` 类实现
   - `repeat_interleave` 广播逻辑详解
   - 推理加速：KV Cache 减少 4-8 倍，推理速度提升 2-3 倍

4. **LoRA（低秩自适应）** - 轻量级微调术
   - 全量微调的沉重负担（7B 模型需要 40GB 显存）
   - 低秩分解数学：ΔW = B @ A
   - 参数效率计算：r=16 时仅 0.39% 参数（减少 250 倍）
   - Kaiming 初始化和零初始化策略
   - QLoRA 结合 4-bit 量化：显存需求 40GB → 12GB
   - 完整的 `LoRALinear` 类和 `inject_lora_to_model` 函数

#### 1. **数据增强与正则化技术**（新增 1.8 节）

**1.8.1 数据增强（Data Augmentation）**
- **图像数据增强工具集**：`ImageAugmentation` 类
  - 几何变换：随机裁剪、水平翻转、旋转、平移

  - 训练/测试模式区分：训练时强增强，测试时只标准化

- **Cutout**：随机遮挡图像的一部分
  - 完整的 `Cutout` 类实现
  - 强制模型学习全局特征，而非依赖局部特征

- **Mixup**：混合两张图像和它们的标签
  - 完整的 `mixup_data` 和 `mixup_criterion` 函数实现
  - 公式：x_new = λ * x_i + (1 - λ) * x_j
  - 对抗性训练技巧

**1.8.2 正则化技术**
- **L1 和 L2 正则化**：
  - 完整的 `l1_regularization` 和 `l2_regularization` 函数
  - L1：鼓励稀疏权重，特征选择
  - L2：防止权重过大，平滑决策边界
  - 完整的 `train_with_regularization` 训练函数

- **Dropout — 随机失活**：
  - 完整的 `MLPWithDropout` 类实现
  - 训练时随机失活，推理时自动处理
  - 集成学习思想：训练多个"子网络"
  - 失活率：通常 0.2 ~ 0.5

- **Early Stopping — 早停**：
  - 完整的 `EarlyStopping` 类实现
  - 参数：patience、min_delta、mode
  - 防止过拟合的有效策略

#### 2. **全新附录 A：斗气修炼心法 — 优化算法与损失函数**

**A.1 优化算法详解**

- **A.1.1 梯度下降（Gradient Descent）**：
  - 手动实现 `gradient_descent_step`
  - 公式：θ_new = θ_old - lr * ∇L(θ)
  - 三种变体对比：批量 GD / SGD / Mini-batch GD

- **A.1.2 动量（Momentum）**：
  - 完整的 `MomentumOptimizer` 类实现
  - 公式：v_t = γ * v_{t-1} + lr * ∇L(θ)
  - 惯性助推：加速收敛，减少震荡

- **A.1.3 AdaGrad — 自适应学习率**：
  - 完整的 `AdaGradOptimizer` 类实现
  - 公式：G_t = G_{t-1} + (∇L(θ))²
  - 问题：学习率持续递减

- **A.1.4 RMSprop — AdaGrad 的改进**：
  - 完整的 `RMSpropOptimizer` 类实现
  - 指数移动平均代替累积和
  - 适用于 RNN、在线学习

- **A.1.5 Adam — 自适应矩估计（最常用）**：
  - 完整的 `AdamOptimizer` 类实现
  - 公式：结合一阶矩（梯度均值）和二阶矩（梯度方差）
  - 默认超参数：lr=0.001, β1=0.9, β2=0.999
  - PyTorch 使用示例

- **A.1.6 AdamW — Adam + 权重衰减**：
  - 权重衰减与梯度解耦
  - 推荐用于大模型、预训练
  - Adam vs AdamW 对比表

**A.2 损失函数详解**

- **A.2.1 回归问题损失**：
  - **均方误差（MSE）**：对大误差敏感
  - **平均绝对误差（MAE）**：对异常值不敏感
  - **Huber Loss**：结合 MSE 和 MAE 的优点
    - 误差小时用 MSE（平滑）
    - 误差大时用 MAE（鲁棒）

- **A.2.2 分类问题损失**：
  - **交叉熵损失（Cross-Entropy）**：
    - 完整的 `manual_cross_entropy_loss` 实现
    - 概率分布差异的衡量
  - **二元交叉熵损失（Binary Cross-Entropy）**：
    - 适用于二分类和多标签分类
  - **Focal Loss**：
    - 完整的 `FocalLoss` 类实现
    - 解决类别不平衡问题
    - 聚焦困难样本：(1 - p_t)^γ

- **A.2.3 对抗损失（Adversarial Loss）**：
  - GAN 损失函数
  - 判别器和生成器的目标
  - 完整的 `gan_loss` 函数实现

**A.3 修炼心得总结**
- 优化算法对比表（SGD/Momentum/AdaGrad/RMSprop/Adam/AdamW）
- 损失函数对比表（MSE/MAE/Huber/CrossEntropy/Focal）
- 适用场景和特点
