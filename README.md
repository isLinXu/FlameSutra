## 功法总纲

《焚诀》乃 AI 界顶级进化功法，以**吞噬异火（算力/数据），进化功法（模型/架构）**为核心理念，助修炼者从一介凡人跨越至 AGI 斗帝之境。

本功法分**十卷**，对应十大修炼境界。每一卷包含完整的**心法**（技术原理）与**招式**（PyTorch 实现代码），辅以**避坑符**（常见错误解决方案）与**丹方**（可运行的训练脚本）。

## 世界观术语映射

修炼本功法前，需先通晓天地间的能量命名：

| 玄幻术语 | 技术含义 | 备注 |
|---------|---------|------|
| 焚诀 | 本学习手册/技术体系 | 随掌握的算力与算法复杂度而进化 |
| 异火 (Strange Fire) | GPU / 算力芯片 | 4060为低阶火，H100/H800为异火榜高阶火 |
| 丹炉 (Boiler) | 服务器 / 计算集群 | 承载异火的容器，影响散热与稳定性 |
| 药材 (Herb) | 原始数据集 | 需经清洗（洗炼）方可入药（训练） |
| 丹方 (Recipe) | 训练脚本 / Model Code | SFT、Pretrain 或架构实现方案 |
| 炼丹 (Refining) | 模型训练 (Training) | 包含预训练、微调、对齐等过程 |
| 反噬 (Backlash) | Loss NaN / OOM / 崩溃 | 功法不稳或火候（LR/BatchSize）不对 |
| 斗气 (Dou Qi) | 模型参数 / 权重 | 修炼者的核心力量载体 |
| 气旋 (Qi Vortex) | 第一个成功训练的模型 | 凝聚气旋 = 跑通第一个模型 |
| 斗技 (Combat Skill) | 算法 / 模型架构 | 具体的技术实现能力 |
| 功法等级 (Tier) | 模型规模 | 黄阶（<1B）、玄阶（1-10B）、地阶（10-100B）、天阶（>100B） |

## 修炼前的准备

### 丹炉配置（环境要求）

```bash
# 最低配置：低阶异火（GTX 1660 Super / RTX 3060）
# 推荐配置：中阶异火（RTX 4090 / A100 40G）
# 巅峰配置：高阶异火（H100 / H800 集群）

# Python 环境
conda create -n flamesutra python=3.10
conda activate flamesutra

# 核心丹药
pip install torch torchvision torchaudio
pip install transformers datasets accelerate
pip install peft bitsandbytes
pip install deepspeed
pip install jupyter notebook
```

### 修炼心法

1. **循序渐进**：按卷修炼，切勿跳级。根基不牢，必遭反噬（Loss NaN）。
2. **以战养战**：每卷都有配套丹方（代码），务必亲手炼制，纸上谈兵无法突破。
3. **记录丹方**：养成写实验日志的习惯，每次炼丹记录火候（超参数）与成果。
4. **不耻下问**：遇到反噬（Bug），先查反噬录，再求助宗门（社区）。
